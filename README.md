# Imperva (imperva)

Imperva (a Thales company) is a cybersecurity company providing cloud-based and on-premises application security, data security, and network security solutions. Their developer APIs cover cloud WAF management, DDoS protection, API security, advanced bot protection, and data security fabric for enterprises. Developers can manage sites, policies, configurations, and analytics programmatically via REST APIs authenticated with API keys, and Imperva provides a Python SDK for SecureSphere on-premises management along with a Terraform provider and CLI for infrastructure automation.

## APIs

- **Imperva Cloud Application Security API** - REST API for managing cloud WAF, DDoS protection, CDN, bot management, and API security policies. [Documentation](https://docs-cybersec.thalesgroup.com/bundle/cloud-application-security/page/apiv2/cloud-api.htm)
- **Imperva SecureSphere Open API** - REST API for managing on-premises SecureSphere MX appliances, security policies, and configurations. [Documentation](https://docs-cybersec.thalesgroup.com/bundle/api-docs/page/api/api-overview.htm)
- **Imperva API Security** - API for managing API visibility, threat detection, and access policy enforcement across REST and GraphQL APIs. [Documentation](https://docs-cybersec.thalesgroup.com/bundle/basic-api-security/page/80414.htm)
- **Imperva Data Security Fabric API** - API for automated deployment and configuration of data security monitoring and auditing across cloud and on-premises repositories. [Documentation](https://docs-cybersec.thalesgroup.com/bundle/api-docs/page/api/api-overview.htm)
